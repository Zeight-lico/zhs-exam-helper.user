## 智慧树考试自动答题助手 说明
### 简介
智慧树考试自动答题助手是一款运行在 Tampermonkey/Violentmonkey 上的用户脚本，专为 智慧树（zhihuishu.com） 在线考试平台设计，能够实现 全自动读题 → AI答题 → 自动选填 → 自动跳转 的一键式考试辅助。

### 核心功能
🤖 多AI供应商支持 内置接入 DeepSeek 、 通义千问（阿里） 、 智谱 GLM 、 Kimi（月之暗面） 、 硅基流动 、 OpenAI 、 豆包（字节） 等主流大模型API，也支持自定义API地址，满足不同用户的模型偏好。

📖 智能题目读取 通过 Hook 智慧树页面的 Vue 框架实例，从组件的 shadowDom 中精准提取题目文本和选项内容，兼容多版本 Vue（2.5.0 / 2.5.2），并有多层回退机制确保题目读取成功率。

✅ 多题型支持

- 单选题 ：AI 返回如 A ，自动选中对应选项
- 多选题 ：AI 返回如 A,C,D ，自动依次选中多个选项
- 判断题 ：AI 将正确/是的判断映射为 A ，错误/否的判断映射为 B
🛡️ 反检测机制

- Hook setInterval / setTimeout ，拦截平台的反作弊检测代码
- 屏蔽 blur 、 focusout 、 mouseleave 等切屏监听事件
- 清除 onblur 事件，防止切出页面被检测
⏱️ 模拟真人作答 每次选题后随机等待 3~7秒 再点击下一题，避免因操作过快触发平台异常检测。

🔄 容错重试 API 调用失败时自动重试（最多3次），应对网络波动或服务不稳定。

🎛️ 可拖拽控制面板 右侧悬浮面板支持鼠标拖拽移动、折叠/展开，界面简洁直观，包含：

- AI供应商切换下拉框（含获取Key链接）
- API Key 输入框（本地加密保存）
- 测试连接按钮，验证API是否可用
- 实时日志窗口（支持一键复制日志）
- 运行状态指示（就绪 / 运行中 / 已停止）
- 答题进度与成功/失败计数
### 使用方式
1. 安装 Tampermonkey 或 Violentmonkey 浏览器扩展
2. 导入本脚本
3. 进入智慧树考试页面（URL 匹配 *.zhihuishu.com/*exam* ）
4. 选择一个 AI 供应商，点击"获取Key"跳转对应平台注册并获取 API Key
5. 在面板中输入 API Key 并点击保存
6. 可选：点击"测试连接"验证 API 是否正常
7. 点击"开始自动答题"，脚本将自动逐题完成考试
8. 如需中途停止，点击"停止答题"即可

### 注意事项
- 本脚本仅供学习研究使用，请遵守智慧树平台的用户协议和相关法律法规
- API Key 存储在浏览器本地，不会上传到任何第三方服务器
- 建议使用有免费额度的供应商（如硅基流动），降低使用成本
